## 📌 Overview

HR Management System is a web application designed to help companies manage employees, attendance, payroll, departments, leave requests, and more.

The system provides different dashboards and permissions for Admin, HR, and Employees.

---

## 🚀 Features

### 👨‍💼 Admin
- Manage role-based access control
- View, delete users, and change user roles
- Manage employees
- Manage departments
- View attendance records
- Manage payroll
- Manage leave requests

---

### 👩‍💻 Employee
- Check in / Check out
- View attendance history
- Submit leave requests
- View salary and payroll information
- Access personal dashboard

---

### 🧑‍💼 HR
- Approve or reject leave requests
- Monitor employee attendance
- Manage employee data
- Access HR and employee features

---

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/HR-SystemProject/HRSystem.git
```

### Install frontend dependencies

```bash
cd frontend
npm install
```

### Install backend dependencies

```bash
cd backend
npm install
```

---

## ▶️ Run the Project

### Start Backend

```bash
npm run server
```

### Start Frontend

```bash
npm run dev
```

---

## 📁 Environment Variables

A `.env.example` file is included in the project.

Create a `.env` file inside the backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
